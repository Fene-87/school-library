# 📖 School Library <a name="about-project"></a>

**School Library** is a ruby application whose aim is to serve as a library tool that records what books are in the library and who borrows them This app allows the user to add new students or teachers, new books, and save records of who borrowed which book and when.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

  <ul>
    <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
  </ul>

### Key Features <a name="key-features"></a>

- **[Add new teachers and students.]**
- **[Add new books.]**
- **[Keep records of books borrowed, by whom, and when.]**

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Ruby
- Ruby CLI

### SetUp

Clone this Repository to your desired folder:

``` sh
cd my-folder
git clone git@github.com:Fene-87/school-library.git
```
### Install 
Install this project with:
 
``` sh
cd OOP-schho-library
bundle install
```

### Usage
To run the project, execute the following command:
``` sh
ruby person.rb
```
